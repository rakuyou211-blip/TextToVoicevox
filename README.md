# テキスト抽出 → VOICEVOX ツール（オフライン / Windows・macOS対応）

**作者: RAKUYOU211**（GitHub: [rakuyou211-blip](https://github.com/rakuyou211-blip)）— 本ツールのオリジナル作成者

📦 [ダウンロード（最新版）](https://github.com/rakuyou211-blip/TextToVoicevox/releases/latest) ｜ 🐦 [X（旧Twitter）](https://x.com/rakuyou211) ｜ ▶️ [YouTube](https://www.youtube.com/@rakuyou211)

PDF・画像（写真）から **テキストだけを取り出し**、VOICEVOX で読み上げ・音声化できる形に整えるツールです。
**Windows と macOS の両方で動作**し、ネット接続なしで使えます。

- OCR … OS標準のオフラインOCRを使用
  - Windows: Windows標準OCR（Windows.Media.Ocr）
  - macOS: Apple Vision フレームワーク（写真.appの文字認識と同じエンジン）
- PDF解析 … 同梱ライブラリ（pypdfium2）
- 音声合成 … ローカルの VOICEVOX エンジン

---

## できること

- **PDF** からテキスト抽出
  - テキスト層がある PDF → そのまま高精度に抽出
  - スキャン画像の PDF → 自動で OCR
- **画像（PNG / JPG / BMP / TIFF / WEBP / GIF）** から OCR でテキスト抽出
- 抽出テキストを **VOICEVOX 向けに整形**
  - 文ごとに改行(VOICEVOX のテキスト読み込みで 1 行 = 1 ブロックになる)
  - 文字間の余計な空白を自動除去
- 結果を画面で **手直し** してから出力
- 出力方法
  1. **`.txt` 保存** → VOICEVOX の「ファイル → テキスト読み込み」で取り込み
  2. **クリップボードにコピー**
  3. **WAV 音声を直接生成**（VOICEVOX エンジン起動時。話者・話速を選択可能）

### 便利機能
- **クリップボードOCR**：スクリーンショット等で画像をコピーしてボタンを押すと、その場で文字起こし（結果に追記）。
- **▶ 試聴(カーソル行)**：結果テキストでカーソルのある行を、選択中の話者・話速ですぐ再生（エンジン接続時）。
- **一括置換**：結果テキストの OCR 誤り（例: ボ→ホ、て→で）をまとめて修正。`Ctrl+Z`（Macは`Cmd+Z`）で取り消し可。
- **エンジンURL変更**：ポートを変えている場合は画面の URL 欄を書き換えて接続確認。
- **設定の記憶**：出力形式・DPI・話者・話速・ウィンドウサイズなどを次回も自動復元（`settings.json`）。

## 初回セットアップ（1回だけ・ネット必要）

### Windows
1. `setup.bat` をダブルクリック
   - Python の仮想環境（`venv`）を作り、必要ライブラリを入れます

### macOS
1. `setup.command` をダブルクリック
   - 「開発元を検証できない」と出た場合は、右クリック →「開く」で実行
   - Python の仮想環境（`venv_mac`）を作り、必要ライブラリを入れます
   - `python3` が無い場合はターミナルで `xcode-select --install` を実行

> ※ この初回だけネット接続が必要。以降はオフラインで動作します。

## 使い方

1. 起動
   - Windows: **`起動.bat`** をダブルクリック
   - macOS: **`起動.command`** をダブルクリック
2. PDF / 画像を登録（次のどれでも）
   - **ファイルやフォルダを画面にドラッグ＆ドロップ**（フォルダ内の対応ファイルをまとめて追加）
   - 「ファイル追加」/「フォルダ追加」ボタン
3. オプションを選ぶ
   - **出力形式**：通常は「文ごとに改行（VOICEVOX推奨）」
   - **PDF処理**：「自動」でOK。文字化けするPDFは「常にOCR」
   - **小説向け**：改行で文がぶつ切りになる場合は「改行で途切れた文を連結」にチェック
4. **「▶ テキスト抽出 実行」**
5. 結果を確認し、必要なら **画面上で修正**（OCRは濁点などを間違えることがあります）
6. VOICEVOX へ
   - **`.txt` 保存** → VOICEVOX で「テキスト読み込み」
   - または **「VOICEVOX起動」→「エンジン接続確認」→ 話者選択 →「🔊 音声を生成(WAV)」**

## OCR 精度について

- OS標準のオフラインOCRを使用しています。**印刷された文字**はかなり正確です。
  - macOS の Vision は特に認識精度が高めです。
- 低解像度・手書き・装飾フォントは誤認識（特に **濁点／半濁点**：で→て など）が出ることがあります。
  - 「画像前処理（精度向上）」を ON（既定）にすると改善します。
  - PDF は「OCR解像度(DPI)」を上げると精度が上がります（その分遅くなります）。
- 仕上げは **結果欄での手修正** が確実です。

| `venv/` / `venv_mac/` | Python 仮想環境（セットアップで生成） | Win / Mac |

## トラブルシューティング

- **起動しない** → `デバッグ起動.bat`（Mac: `デバッグ起動.command`）でエラー内容を確認
- **OCRが空になる（Windows）** → Windows設定 →「時刻と言語」→「言語と地域」→ 日本語の「言語のオプション」に
  **基本的な入力（OCR）** が入っているか確認
- **OCRが空になる（macOS）** → `setup.command` を再実行して `pyobjc-framework-Vision` が入っているか確認
- **音声生成できない** → 先に VOICEVOX を起動し、「エンジン接続確認」が「接続OK」になっているか確認
  - ポートを変えている場合は画面の **URL欄** を変更（例: `http://127.0.0.1:50022`）
- **VOICEVOX起動ボタンが効かない** → VOICEVOX を標準の場所にインストールしているか確認
  - Windows: `%LOCALAPPDATA%\Programs\VOICEVOX`
  - macOS: `/Applications/VOICEVOX.app`
- **画面が真っ白（macOS）** → 古い Python 付属の Tk (8.5) が原因。Tk 8.6 以降を含む Python
  （[python.org](https://www.python.org/downloads/) の公式版など）を入れてから `setup.command` を再実行

## 音声（VOICEVOX）の利用について

このツールは VOICEVOX 本体を同梱していません。ローカルで動作する **VOICEVOX エンジン**に
テキストを渡して音声を生成する「クライアント」です。VOICEVOX 本体は各自でインストールしてください。

**生成した音声を公開・配布する場合は、VOICEVOX および各キャラクターの利用規約に従ってください。**
多くのキャラクターでは、音声を用いた作品の公開時に **クレジット表記**（例:「VOICEVOX:四国めたん」）が
必要です。詳細は [VOICEVOX 公式サイト](https://voicevox.hiroshiba.jp/) と各キャラクターの利用規約を確認してください。
