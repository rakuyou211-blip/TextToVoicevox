**本や書類を「ずんだもん」たちに読み上げてもらう、無料のデスクトップアプリです。**

PDF・画像・txt・docx・epub から文字を取り出して、あなたのパソコンの中の VOICEVOX で朗読・音声化します。文字を目で追うのがつらいときに、耳で聴くという選択肢を。

これは「落葉いの道具箱 🍂」のひとつです。ひとりで作っているので、まだ粗いところもあります。それでも、小さな困りごとに手作りの道具を、というつもりで置いています。

## こんなときに

- 分厚いPDFの資料や本を、**家事をしながら耳で聴きたい**
- 目が疲れていて、これ以上**画面の細かい文字を追いたくない**
- **ディスレクシア**などで、文字を読むより聴くほうが頭に入りやすい
- スキャンした書類・写真に撮った紙の文字を、**音声にして確認したい**
- 小説を**ずんだもんの声**で朗読させて、寝る前に流したい
- 字幕（.srt）付きの音声がほしい
- **ずんだもん達の掛け合い動画**を作りたい（行ごとに話者を分けて、音声も字幕も一括で書き出し）

「読む」がしんどいときに、「聴く」に切り替えるための道具です。

## できること

- **文字の取り出し**：PDF・画像（PNG/JPEG等）・txt・docx・epub に対応
- **オフラインOCR**：画像やスキャンPDFの文字を、OSの標準エンジンで読み取り
  （Windows = Windows.Media.Ocr / Mac = Apple Vision。ネットに送りません）
- **VOICEVOXで音声化**：話者・話速・音高・抑揚・音量を調整して WAV / M4A / MP3 で保存
- **オーディオブック出力（M4B）**：全文を1ファイルにまとめ、章見出し（第◯章など）から自動でチャプターを埋め込み。Apple Books等で「本」として頭出しできます
- **声サンプル試聴**：話者を選ぶとき「🔊 声を聴く」でキャラの公式サンプルを再生
- **読み確認**：試聴すると「どう読んだか」をカナで表示。誤読は読み方辞書ですぐ直せます
- **＃メモ行**：行頭に `#` を付けた行は読み上げから除外（台本メモに）
- **連続再生＋しおり＋一時停止**：長い本も途中から聴き直せます（スペースキーで一時停止/再開）
- **生成のキャンセル**：長い本の音声生成を途中で止められます（Esc）
- **行ごとに話者を変える**：ナレーターとセリフで声を分けるような使い方も
- **字幕（.srt）出力**：結合音声に合わせた字幕を書き出し
- **読み方辞書**：固有名詞などの読み間違いを登録して直せます
- **折り返しを賢く連結**：画像OCRでは行の座標を使って、折り返しで割れた1文を確実につなぎ直します（見出し・箇条書き・別段落は尊重、誤連結なし）
- **青空文庫のルビ除去・Web小説の「漢字(かんじ)」型ルビ除去**
- **画面キャプチャのノイズ除去**：ニュースやSNS埋め込みのスクショから、時刻・局ロゴ・SNSハンドル・矢印などの“映像内オーバーレイ文字”を除いて本文だけ読み上げ（既定ON・保守的判定）
- **映像内ラベルの除去**：ニュース画像OCRに残る局ロゴ・番組名・日時・カテゴリ表示などの短い日本語ラベル（例「MBSニュース」「国内」）を、行の座標・文字サイズから本文でないと判定して除きます（Windows/Mac両対応・上のノイズ除去と共通ON/OFF・本文は必ず残す保守的判定）
- **OCR誤字の自動補正**：OCRが取り違えやすい同形文字（`サ一ビス`→`サービス`・`卜ヨタ`→`トヨタ`・`2O26`→`2026` など）を前後の文脈で確定できるときだけ直します（既定ON・OCR結果のみ対象。テキストファイルには触れません）
- **読めない記号の展開**：`①`→`1`・`㈱`→`株式会社`・`50㎡`→`50平方メートル` など、VOICEVOXが読み飛ばす記号を読みに展開（「全角英数→半角」オプションと連動）
- **ダークモード・文字サイズ変更・本文検索・主要操作のショートカット**（抽出=Ctrl/⌘+Return・生成=Ctrl/⌘+G・試聴=Ctrl/⌘+P・停止=Esc など）
- **CLI（コマンド）モード**：GUIなしで一括変換（自動化・上級者向け）

## 使うまでの順番（ここ大事）

このアプリは単体では音を出せません。**先に VOICEVOX を用意する**必要があります。順番を守ってください。

### ① まず VOICEVOX を入れて、起動しておく

音声を作っているのは VOICEVOX 本体です。このアプリには**同梱していません**。

1. [VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/) から VOICEVOX をダウンロードしてインストール
2. VOICEVOX を**起動したままにする**（起動していると `http://127.0.0.1:50021` で待ち受けます）

> VOICEVOXが起動していないと、このアプリは「エンジンに接続できません」となります。音声化のときは必ず起動しておいてください。

### ② Python を入れる

配布物は Python のスクリプトです。パソコンに Python が必要です。

- [python.org](https://www.python.org/downloads/) から **Python 3.10 以降**を入れるのがおすすめです
- Windowsのインストーラでは、最初の画面の **「Add Python to PATH」にチェック**を入れてください
- （画面表示に tkinter を使います。python.org版のWindows/Mac標準インストーラなら最初から入っています）

<!-- TODO: 動作確認済みの最低Pythonバージョンを確定したら「3.10以降」の数字を見直す -->

### ③ このアプリを入れる

1. 上の「⬇ ダウンロード」から `TextToVoicevox_v1.11.0.zip` を落として、**解凍**する
2. **Windows**：フォルダ内の **`setup.bat`** を実行（初回だけ。venvを作って部品を入れます／ネットが要ります）
   → 終わったら **`起動.bat`** で起動
3. **Mac**：**`setup.command`** を実行（初回だけ）
   → 終わったら **`起動.command`** で起動
4. うまく動かないときは **`デバッグ起動.bat` / `デバッグ起動.command`**（黒い画面にエラーが出るので原因が分かります）

あとは、開いた画面にファイルをドラッグ＆ドロップ → 文字を取り出す → 必要なら直す → 音声を作る、の流れです。

**なぜ安全と言えるのか**：このアプリは中身を**全部ソースで公開（MIT）**しています。`setup.bat` も `main.py` も `core.py` も、このリポジトリで一行ずつ読めます。何をしているか隠していません。不安なら、詳しい人に中身を見てもらってから使っても大丈夫です。

### Windows で警告が出たとき

- **「WindowsによってPCが保護されました」**と出たら → 小さい **「詳細情報」** をクリック → **「実行」** ボタンが出ます
- ダウンロードした `.bat` が反応しない／ブロックされる感じがするときは → そのファイルを右クリック → **プロパティ** → 下のほうの **「ブロックの解除」** にチェック → OK

- `setup.command` や `起動.command` を**普通にダブルクリックすると「開けません」**と出ることがあります
  → その場合は、ファイルを **右クリック（Controlキーを押しながらクリック）→「開く」** → もう一度「開く」で実行できます
  → 「アクセス権がありません」と言われるとき（zip 解凍で実行権限が外れた場合）は、ターミナルで
  `chmod +x "解凍したフォルダのパス"/*.command` を一度実行してからダブルクリックしてください
- それでもダメなら、ターミナルで次を実行してから開いてください（ダウンロードにつく検疫フラグを外します）
  `xattr -dr com.apple.quarantine "解凍したフォルダのパス"`

## コマンド（CLI）で使いたい人へ

GUIを使わず、一括で変換できます。`起動.bat` / `起動.command` と**同じ venv の Python** で実行してください。

    # PDF/画像/テキスト → VOICEVOX向けの整形txtを作る（エンジン不要）
    python cli.py 本.pdf -o 出力フォルダ

    # 音声まで一括生成（VOICEVOX起動が必要）
    python cli.py 本.pdf -o 出力フォルダ --wav --speaker ずんだもん --combine --srt

`python cli.py -h` で全オプション（話速 `--speed`、字幕 `--srt`、ルビ除去 `--paren-ruby`、OCR解像度 `--dpi`、エンジンURL `--url` など）が見られます。

## クレジット・利用について

- 音声合成は **[VOICEVOX](https://voicevox.hiroshiba.jp/)** を使っています。ずんだもん等のキャラクター音声は VOICEVOX のものです
- **生成した音声を公開・配布するときは、VOICEVOX および各キャラクター（ずんだもん等）の利用規約・クレジット表記に必ず従ってください。** 規約は各キャラクターごとに異なります
- **キャラ立ち絵（任意）**：`assets/立ち絵/` に立ち絵PNGを置くと、話者に合わせてキャラが画面右に表示されます。**立ち絵はリポジトリに同梱していません**（公式素材の二次配布を避けるため）。入手先・置き方・クレジットは [`assets/立ち絵/README.md`](assets/立ち絵/README.md) を参照（立ち絵:坂本アヒル 氏／管理:東北ずん子・ずんだもんプロジェクト）
