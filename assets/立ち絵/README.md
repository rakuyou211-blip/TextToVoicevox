# キャラ立ち絵（任意・ローカルのみ・全キャラ対応）

このフォルダに立ち絵PNGを置くと、アプリ右側に**選択中の話者のキャラ**が表示されます。
VOICEVOXの**全43キャラのどれでも**、キャラ名でPNGを置くだけで対応します。
**画像ファイルはリポジトリに含めていません**（`.gitignore` 済み。公式素材の二次配布を避けるため）。
無い場合でもアプリは通常どおり動作します（そのキャラの立ち絵が無ければ枠が空になるだけ）。

## 置き方（かんたん）

**ファイル名 ＝ キャラ名 ＋ `.png`**。たとえば：

| ファイル名 | 表示されるキャラ |
|---|---|
| `ずんだもん.png` | ずんだもん |
| `四国めたん.png` | 四国めたん |
| `春日部つむぎ.png` | 春日部つむぎ |
| `九州そら.png` | 九州そら |

話者名の `（ノーマル）` などのスタイル部分は不要です（キャラ名だけでOK）。
選んだ話者に対応するPNGがあればそのキャラを、無ければ何も表示しません（別キャラを間違って出しません）。

### ファイル名に使えない文字を含むキャラ

`/ \ : * ? " < > |` と空白は、ファイル名から**取り除いて**保存してください（アプリ側も同じ規則で照合します）。

| キャラ名 | 置くファイル名 |
|---|---|
| `小夜/SAYO` | `小夜SAYO.png` |
| `†聖騎士 紅桜†` | `†聖騎士紅桜†.png`（空白を除く） |
| `ナースロボ＿タイプＴ` | `ナースロボ＿タイプＴ.png`（全角なのでそのまま可） |

### 旧バージョンからの引き継ぎ

`zundamon.png` / `metan.png`（ローマ字）はそのまま使えます（自動でずんだもん・四国めたんに割り当て）。

## まばたき・口パク（任意）

同ポーズの差分を置くと動きます：待機中は**まばたき**、試聴・連続再生中は**口パク**（行の話者に合わせてキャラも切替）。

| 追加ファイル | 用途 |
|---|---|
| `<キャラ名>_closed.png` | 口を閉じたポーズ（あると基本ポーズに使う） |
| `<キャラ名>_open.png` | 口を開いたポーズ（口パク用） |
| `<キャラ名>_blink.png` | 目を閉じたポーズ（まばたき用） |

例：`ずんだもん_closed.png` / `ずんだもん_open.png` / `ずんだもん_blink.png`。
`_closed` があるときは基本ポーズとして `_closed` を使うので、`_open`/`_blink` と**同ポーズの差分**を置いてください（公式立ち絵PSDの口パク書き出しがそのまま使えます）。差分が無ければ静止画のまま動作します。

透過PNGならサイズは自由（実行時にパネル幅へ自動縮小）。PSDから透過PNGを書き出すには [PSDTool](https://oov.github.io/psdtool)（ブラウザ・Mac可）が便利です。

## クレジットと利用ガイドライン（重要）

立ち絵は**キャラごとに権利者・利用条件が異なります**。使う立ち絵ごとに、各キャラの公式ガイドラインを必ず確認し、必要なクレジットを表記してください。アプリ画面には汎用の `VOICEVOX ／ 立ち絵：各キャラのガイドライン準拠` を表示しています（音声の `VOICEVOX:話者名` クレジットは生成完了時に別途案内します）。

- 各キャラの規約・素材配布先は VOICEVOX 公式サイトのキャラクター一覧から辿れます：https://voicevox.hiroshiba.jp/
- 東北ずん子・ずんだもん系（ずんだもん／四国めたん／東北ずん子 ほか）：https://zunko.jp/guideline.html
  - 例）ずんだもん・四国めたんの立ち絵素材（坂本アヒル 氏）：
    - ずんだもん：https://seiga.nicovideo.jp/seiga/im11206626
    - 四国めたん：https://seiga.nicovideo.jp/seiga/im10791276

> このアプリは立ち絵を同梱・再配布しません。表示に使う画像はご自身で入手し、各キャラのガイドラインの範囲でご利用ください。
