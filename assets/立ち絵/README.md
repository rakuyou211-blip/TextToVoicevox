# キャラ立ち絵（任意・ローカルのみ）

このフォルダに立ち絵PNGを置くと、アプリ右側に**選択中の話者のキャラ**が表示されます。
**画像ファイルはリポジトリに含めていません**（`.gitignore` 済み。公式素材の二次配布を避けるため）。
無い場合でもアプリは通常どおり動作します（キャラ枠が出ないだけ）。

## 置き方

| ファイル名 | キャラ | 用途 |
|---|---|---|
| `zundamon.png` | ずんだもん | 基本の立ち絵（これだけでもOK） |
| `metan.png` | 四国めたん | 同上 |
| `zundamon_closed.png` / `_open.png` / `_blink.png` | ずんだもん | 任意：口閉じ/口開き/まばたき（同ポーズの差分） |
| `metan_closed.png` / `_open.png` / `_blink.png` | 四国めたん | 同上 |

透過PNGならサイズは自由（実行時にパネル幅へ自動縮小）。話者名に「ずんだもん」「四国めたん（めたん）」を含むと対応画像に切り替わり、その他の話者では既定でずんだもんを表示します。

**差分フレームを置くと動きます**：待機中は まばたき、試聴・連続再生中は口パク（行の話者に合わせてキャラも切替）。`_closed` があるときは基本ポーズとして `_closed` を使うので、`_open`/`_blink` と**同ポーズの差分**を置いてください（公式立ち絵PSDの口パク書き出しがそのまま使えます）。無ければ静止画のまま動作します。

## 公式無料素材（坂本アヒル氏）の入手先

- ずんだもん立ち絵素材：https://seiga.nicovideo.jp/seiga/im11206626
- 四国めたん立ち絵素材：https://seiga.nicovideo.jp/seiga/im10791276
- 作者：坂本アヒル 氏（https://twitter.com/sakamoto_ahr）
- 管理：東北ずん子・ずんだもんプロジェクト（SSS合同会社）
- 利用ガイドライン（必読）：https://zunko.jp/guideline.html

PSDから透過PNGを書き出すには [PSDTool](https://oov.github.io/psdtool)（ブラウザ・Mac可）が便利です。
`_default` 系の書き出しPNGを `zundamon.png` / `metan.png` にリネームしてここへ置いてください。

## クレジット

- **音声（必須）**：`VOICEVOX:ずんだもん` / `VOICEVOX:四国めたん`
- **立ち絵（任意）**：`立ち絵:坂本アヒル`

アプリ画面にも `VOICEVOX / 立ち絵:坂本アヒル` を常時表示しています。
